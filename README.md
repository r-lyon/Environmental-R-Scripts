# Environmental R Portfolio

A collection of R projects demonstrating data analysis, visualization, and environmental monitoring work.

---

## 🌍 Featured Projects

### 🗺️ [Sample Location Webmap](https://github.com/r-lyon/sample-location-webmap)
Interactive Leaflet webmap showing environmental sampling locations in Upper Sandia Canyon.  
**Tools:** R, `leaflet`, `sf`, `dplyr`  
**Highlights:**
- Loads and cleans geospatial data from Intellus 2024 exports
- Displays interactive popups and color-coded sample types
- Exports an interactive HTML map for online viewing

🔗 **View Code:** [sampled_locations_Intellus_2024.R](https://github.com/r-lyon/sample-location-webmap/blob/main/code/sampled_locations_Intellus_2024.R)

### 💧 [Exceedance Screening and Visualization](https://github.com/r-lyon/Environmental-R-Scripts/tree/main/exceedance)
Script collection for screening water quality data against state criteria and generating exceedance ratio plots.  
**Tools:** R, `ggplot2`, `dplyr`, `cowplot`

### 🌿 [Wetland Performance Analysis](https://github.com/r-lyon/Environmental-R-Scripts/tree/main/wetland)
LiDAR and hyperspectral-based analysis of wetland geomorphic and vegetation changes.  
**Tools:** R, `terra`, `lidR`, `ggplot2`
